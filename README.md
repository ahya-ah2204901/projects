# Projects

### STARX – Worker Health Monitoring System

Real-time worker monitoring system that collects physiological and environmental sensor data using ESP32 hardware, Firebase cloud services, and a deep learning–based heat stress prediction pipeline.

---

### BirthLink – SMS-Based Maternal Emergency Coordination System

Hackathon project focused on connecting pregnant women in crisis settings with nearby volunteers using SMS-based communication without requiring internet access.

---

### AccelAid – Post-Crisis Education Reconstruction Platform

AI-powered learning platform designed to support post-crisis education through personalized study plans, assessments, and progress tracking.

---

### Mizăn – Workload Management Web Application

Full-stack web application enabling students, instructors, and coordinators to manage assessments, comments, and workload summaries efficiently.

---

### YalaPay – Financial & Cheque Management System

Financial management application supporting customer, invoice, payment, and cheque processing workflows with Firebase and SQLite integration.

---

### Qatar History Website

Educational website developed for a Qatar History course, showcasing historical sites and cultural landmarks in Qatar through an interactive web interface.
